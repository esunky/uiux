Built-in CSS

어떤 이유로 브라우저가 '퍼블리셔가 작성한 CSS'를 읽지 못한다면 브라우저자체에 내장된 기본스타일로 웹페이지에 적용하여 출력한다.

css reset(태그의 초기화)이 필요한 이유

웹브라우저마다 내장된 기본 스타일이 완전히 똑같지 않아 크로스브라우징(여러가지페이지에서 거의 비슷하게 구현)을 구현하기 어렵기 때문에 자주 사용하는 태그들에 한해 기본스타일값을 초기화(0,zero)한다.

약식초기화

CSS 코딩의 들어가면서 페이지에 속하는 모든 요소들의 여백을 초기화 시키는 작업 - *{margin:0; padding:0;}
